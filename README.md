# ✨ StyleMate – AI Fashion Stylist

StyleMate is a smart fashion recommendation web app that suggests outfit ideas based on user preferences like gender, age, body type, skin tone, and occasion.

It provides direct links to popular fashion platforms like Amazon, Myntra, H&M, Zara, and Meesho for easy shopping.

---

## 🚀 Features

- 🎯 Personalized outfit recommendations
- 👗 Multiple categories: Tops, Pants, Shoes, Accessories
- 🧠 Smart filtering based on:
  - Gender (Female, Male, Kids)
  - Age Group (Kid, Teen, Adult)
  - Body Type (Petite, Hourglass, Pear, Apple, Athletic, Plus Size)
  - Skin Tone (Fair, Medium, Olive, Dark)
  - Occasion:
    - Casual
    - Party
    - Traditional
    - College
    - Gym
    - Beach
- 🔗 Direct shopping links for each brand
- 🎨 Modern UI with glowing buttons and aesthetic design
- ⚠ Input validation (requires all selections before recommendations)

## ▶ How to Run

1. Download or clone the repository
2. Open the folder in VS Code
3. Run `index.html` using Live Server
4. Click **Get Started**
5. Select all filters
6. Click **Get Styled**
7. Explore outfit links from different brands

---

## 💡 How It Works

- User selects all required filters
- JavaScript validates inputs
- Based on occasion, curated product links are generated
- Cards for each brand are displayed dynamically
- Each button redirects to the respective shopping platform
